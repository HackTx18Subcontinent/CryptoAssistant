Crypto Assistant

Crypto Assistant HACKTX 2018 Teza AltData

Enter the cryptocurrency that you want to get insights of using sentiment analysis. We intended to use Google Home but didn't finish. For example you would ask "Hey Google, how is Bitcoin doing today?" and Google would reply "For Bitcoin I analyzed 50 articles. I found 42 neutral articles, 3 positive articles, and 5 negative articles. The overall sentiment is looking Negative."

Processing takes about 4 seconds.
